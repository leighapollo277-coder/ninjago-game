## Project Context
The `NinjagoGameApp.jsx` file implements a educational game for learning Chinese characters through a Ninjago-themed interface. Key systems include:
- **Speech Synthesis**: Uses the Web Speech API for Cantonese pronunciation.
- **Battle Mode**: A "Combat" system where energy is consumed on mistakes.
- **Training Mode**: A specialized view for reviewing missed words.
- **Logging**: Integration with Google Sheets for tracking progress.
